# Customer Purchase Prediction using Logistic Regression

## About the Project
I built this project to understand how machine learning can be used to predict customer purchasing behavior. Using customer information such as age and estimated salary, I trained a Logistic Regression model to determine whether a customer is likely to purchase a product.
The project covers the complete machine learning workflow, from data preprocessing and feature scaling to model training, prediction, and performance evaluation. It helped me gain practical experience with classification problems and understand how data can be used to support business decisions.

# What I Learned
- Working with real-world datasets using Pandas
- Preparing data for machine learning models
- Splitting datasets into training and testing sets
- Applying feature scaling using StandardScaler
- Building and training a Logistic Regression model
- Evaluating model performance using accuracy metrics
- Making predictions for new customer data

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Logistic Regression

## Project Workflow
1. Load and explore the dataset
2. Prepare features and target variables
3. Split the dataset into training and testing sets
4. Standardize numerical features
5. Train a Logistic Regression model
6. Predict customer purchase decisions
7. Evaluate model accuracy
8. Test the model with new customer inputs

## Business Value
This project demonstrates how machine learning can be applied to customer analytics. By predicting purchasing behavior, businesses can better understand their target audience and make more informed marketing decisions.
